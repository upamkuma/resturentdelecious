# 🍽️ Restaurant App

A modern and responsive **Restaurant Web Application** built with(frontend) basic html, css , javascript , bootstarp and google map integration.  
It allows customers to browse menu items, place orders, and contact the restaurant. Admins can manage menu and orders via a secure dashboard

---

## ✨ Features
- 📖 Interactive and responsive UI  
- 🛒 Add to cart & place orders  
- 🔐 User authentication (Login/Signup)  
- 📋 Admin dashboard to manage menu and orders  
- 📞 Contact form with email integration  
- 📱 Mobile-friendly design
